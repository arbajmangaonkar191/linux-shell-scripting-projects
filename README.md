# Linux Shell Scripting Projects 🚀

This repository contains Linux and Shell Scripting projects that I built while learning DevOps as part of my 90 Days DevOps Challenge.

## 📂 Projects Included

### 1. System Monitoring Script

Features:

* Hostname Check
* Disk Usage Check
* Memory Usage Check
* Nginx Service Monitoring
* Top Memory Consuming Processes
* Server Uptime Monitoring

### 2. User Management Script

Features:

* Create User
* Verify User
* Check Existing Users
* User Home Directory Creation

### 3. Backup Automation Script

Features:

* Backup Files and Directories using Tar
* Compressed Backup (.tar.gz)
* Automated Backup using Cron Jobs
* Timestamp Based Backup Naming

### 4. Disk Monitoring Script

Features:

* Disk Usage Monitoring
* Disk Threshold Alerts
* Warning Messages for High Disk Usage

### 5. Shell Scripting Concepts Practiced

* Variables
* If-Else Conditions
* For Loops
* While Loops
* Functions
* Command Substitution
* Cron Jobs
* Process Monitoring
* Linux Commands

## 🚀 How to Run

Clone the repository:

```bash
git clone https://github.com/arbajmangaonkar191/linux-shell-scripting-projects.git
```

Navigate to the project directory:

```bash
cd linux-shell-scripting-projects
```

Make scripts executable:

```bash
chmod +x *.sh
```

Run a script:

```bash
./script_name.sh
```
